# Convert IHC Marker Data to Numeric Matrix

Convert IHC Marker Data to Numeric Matrix

## Usage

``` r
convertIHCToNumeric(data, markers, id_variable = NULL)
```

## Arguments

- data:

  The dataset

- markers:

  Vector of marker column names

- id_variable:

  Optional ID variable name for row names

## Value

Numeric matrix with markers as columns
